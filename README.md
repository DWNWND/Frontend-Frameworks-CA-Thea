## How to run locally

1. Ensure you have [Node.js](https://nodejs.org/) installed

2. Clone the repo:

```bash
git clone https://github.com/DWNWND/Frontend-Frameworks-CA-Thea.git
```

3. Open the repo with VSCode

4. Install npm dependencies

```bash
npm install
```

5. Run build

```bash
 npm run dev
```
